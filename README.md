# Operation: Remote Agent

A spy-themed interactive game that teaches you to control Claude Code remotely from your iPhone using SSH + tmux + Tailscale.

## Play Online

**[Play Now](https://MasterfullyClaw.github.io/operation-remote-agent/)** — no install needed.

## Run Locally

```bash
git clone https://github.com/MasterfullyClaw/operation-remote-agent.git
cd operation-remote-agent
npm start
```

> Requires Node.js. The `npm start` command uses [`serve`](https://npmjs.com/package/serve) to host the game locally.

## What You'll Learn

- Install and use **tmux** for persistent terminal sessions
- Enable **SSH** on your Mac for remote access
- Set up **Tailscale VPN** for secure peer-to-peer connectivity
- Configure **Termius** on iPhone as your mobile SSH client
- Control **Claude Code** from anywhere

## Tech Stack

- Vanilla JavaScript (ES Modules)
- Zero dependencies — no build step, no bundler
- Pure CSS with terminal green aesthetic
- LocalStorage for game state persistence
